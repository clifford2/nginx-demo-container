Demo Container Error

If you see this page, the nginx demo container has successfully started, but is not working as designed. Further configuration is required.

The most likely cause of this is if the /usr/share/nginx/html directory is not writable, which can happen if you run the container in Kubernetes with a readOnlyRootFilesystem, but did not mount a volume for /usr/share/nginx/html which is writable by UID {NGINX_UID}.

Please see github.com/clifford2/nginx-demo-container/blob/main/deploy/k8s-latest.yaml for an example Kubernetes deployment manifest that addresses this.

Thank you for using nginx.
